# US Population Dashboard

An interactive Streamlit dashboard to explore U.S. population data from 2010 to 2019.

## Features
- Year-based filtering
- Population KPIs and trends
- Charts and tables with CSV download

## Tech Stack
- Python
- Streamlit
- Pandas
- Altair

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
